A website for my favorite new anime, "Delicious in Dungeon", using React, Typescript, and Tailwind CSS. The site features React routes and a custom context.

View on Github Learn More

Utilizing react-router-dom, I implemented routes for each page, ensuring smooth navigation without browser reloads. Additionally, I organized all data related to monsters and recipes in a dedicated constants file for better manageability.

The utilization of the constants file helped with the implementation of a custom React context, enabling dynamic population of the 'Dish' page with appropriate text and images. This approach ensured that the page loads dynamically, adjusting its content based on the dish selected by the user.
